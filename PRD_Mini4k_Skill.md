# 智能影视聚合与纯净种子下载器 (Agent Skill 需求文档)

## 1. 背景与目标
**背景**：用户习惯在电视端（如迅雷云盘/Emby）观看高清影视资源，但其播放设备不支持“杜比视界 (Dolby Vision)”。同时，目标下载网站 (mini4k) 部署了严格的 Cloudflare 防爬虫机制（5秒盾），且需要账号登录。
**目标**：开发一个 OpenClaw Skill，实现“最新影视发现 -> 智能筛选匹配 -> 绕过反爬虫 -> 自动下载纯净种子”的全链路自动化工作流。

## 2. 核心功能模块

### 模块一：豆瓣实时热榜聚合 (Douban Scraper)
*   **触发方式**：每天定时触发（如早晨 8:00），或由 Agent 对话指令实时触发。
*   **数据源**：实时抓取豆瓣官方 API（如 `movie.douban.com/j/search_subjects`），确保数据绝对新鲜，拒绝过时老片。
*   **推送排版要求**：
    1.  **国内版块**：近期最热华语电视剧 Top 10 + 华语电影 Top 10。
    2.  **国外版块**：近期最热欧美/日韩电视剧 Top 10 + 海外电影 Top 10。
    3.  **尝鲜版块**：最新上线的 5 部电影/剧集。
*   **数据字段要求**：必须包含 影片名称、评分、**类型标签**（如喜剧、悬疑）、**连载/完结状态**（如：已完结 全30集 / 连载中 更新至15集）。

### 模块二：Mini4k 智能检索与筛选 (Torrent Fetcher)
*   **触发方式**：用户指定影片名称后，系统自动去 mini4k.com 进行搜索和解析。
*   **筛选过滤规则（核心诉求）**：
    1.  **排除规则（一票否决）**：标题或详情中带有“杜比视界”、“DoVi”、“DV”标签的资源**绝对不下载**（设备不兼容）。
    2.  **清晰度优先级**：优先匹配 `4K` 或 `2160p` 分辨率；若无 4K 资源，则自动降级寻找 `1080p` 资源。
    3.  **字幕要求**：优先选择标注有“简体中文字幕”、“内嵌中字”或“特效中字”的资源。
*   **下载行为**：从目标详情页提取 `.torrent` 文件链接或 `magnet:` 磁力链接，并自动保存到用户指定的本地目录。

### 模块三：Cloudflare 盾绕过与鉴权 (Auth & Bypass)
*   **痛点**：目标网站存在 CF 5秒盾，常规 HTTP 请求（如 axios/fetch）会直接返回 403 Forbidden。
*   **解决方案要求**：
    *   Skill 需集成 `Playwright` 或 `Puppeteer`。
    *   提供“人工介入授权模式”：首次运行时，允许调起带有界面的真实浏览器 (headless: false)，让用户手动打勾过盾并登录账号。
    *   登录成功后，Skill 需自动保存 Cookies / Storage State（如 `auth_state.json`）。后续的静默下载任务直接复用该凭证，实现后台无感下载 (headless: true)。

## 3. 配置与环境要求
*   **下载路径配置**：需提供可配置的参数选项（如 `download_dir`），当前用户默认限制并指定为 `E:\Downloads`。
*   **运行环境支持**：由于用户环境为 Windows 宿主机 + WSL 架构，Skill 的浏览器调起和文件写入需要完美兼容本地挂载路径（如 `/mnt/e/Downloads` 映射）。

## 4. 交互流程示例
1.  **Agent**: "Boss，今天为您准备了最新热门片单：1.《繁花》... 2.《太平年》..."
2.  **User**: "帮我下《太平年》。"
3.  **Agent**: "(后台静默运行 Skill) -> 已在 mini4k 匹配到《太平年 第1-5集 4K 无杜比 简中》种子，已成功下载至 E:\Downloads！迅雷已接管下载。"
